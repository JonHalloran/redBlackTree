Red-Black Tree Tutorial

Rules:

  • Each node is either red or black
  • The root must always be black
  • A red node can never have a red child
  • All nodes start as red

  • Case 0:

    Root is red

    1. Change root color to black

  • Case 1:

    Parent is black

    1. No issue

  • Case 2

    Parent and uncle red.

    1. Change parent and uncle's color to black.
    2. Change grandparent's color to red
    3. Recheck from grandparent

  • Case 3

    Parent is red. Grandparent and uncle are black. Current node on "inside" of parent.

    1. Rotate around parent from direction current node -> parent
    2. Recheck ex-parent, now child of node

  • Case 4

    Parent is red. Grandparent and uncle are black. Current node on "outside" of parent.

    1. Rotate around grandparent from direction parent -> grandparent
    2. Recheck ex-parent, now child of node
    3. Previous grandparent's color goes to red
    4. Parent's color goes to black
